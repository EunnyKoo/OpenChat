# 🎁 크리스마스 테마 오픈 채팅

✅ **[눈 내리는 채팅방]** 크리스마스 테마에 맞춰 눈 내리는 화면 적용

✅ **[배경화면 색 설정]** 사용자가 원하는 채팅방 배경화면 색 지정 가능

✅ **[아이디 중복 확인]** 아이디 중복시 오픈채팅 입장 불가능

✅ **[디엠 기능]** 지정한 사용자에게 디엠 메세지 전송 가능

✅ **[채팅방 종료]** 채팅방 나가는 사용자 아이디 확인 가능 

- **개발 기간** : 2023-12-21 ~ 2023-12-24(4일)
<br />
